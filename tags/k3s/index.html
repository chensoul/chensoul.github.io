通过k3sup安装k3s

2024-05-09
Kubernetes

k3sup是一个支持在PC、虚拟机、ARM设备上安装k3s的工具，官方网站：https://k3sup.dev/

安装k3sup

在线安装：

curl -sLS https://get.k3sup.dev | sh
sudo install k3sup /usr/local/bin/

离线安装，下载地址：https://github.com/alexellis/k3sup/releases
