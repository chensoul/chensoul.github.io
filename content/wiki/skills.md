[Claude-to-IM-skill](https://github.com/op7418/Claude-to-IM-skill) 安装后可以将你的 Claude Code 和 Codex 连接到任何聊天软件，实现远程控制

[Video-Wrapper-Skills](https://github.com/op7418/Video-Wrapper-Skills) 自动为你的视频添加类似“小Lin说”那样的解释特效，支持高亮卡片、关键词卡片、作者介绍、社交媒体卡片等多种组件

[Youtube-clipper-skill](https://github.com/op7418/Youtube-clipper-skill)帮你从 YouTube 下载长视频，并在分析视频后给出“长剪短”的方案，同时会自动添加中英双语字幕。

[Document-illustrator-skill](https://github.com/op7418/Document-illustrator-skill) 自动分析文档并生成配图，支持 16:9 和 4:3 比例。它会按照你的分段生成多张配图，并支持多种风格选择。我常用的封面图就是用这个生成的。

[Humanizer-zh](https://github.com/op7418/Humanizer-zh) 去除 AI 生成文档中常见的“AI 味”表述，采用两段式处理，帮你提升 AI 生成内容的观感。

[NanoBanana-PPT-Skills](https://github.com/op7418/NanoBanana-PPT-Skills) 基于 AI 自动生成高质量 PPT 图片和视频的强大工具，支持智能转场和交互式播放，带有非常顺滑的转场动效。

[baoyu-post-to-wechat](https://github.com/jimliu/baoyu-skills/baoyu-post-to-wechat) 公众号发布技能，支持API和浏览器两种方式发布

[wechat-article-writer](https://github.com/iamzhihuix/happy-claude-skills/wechat-article-writer) 4步流程帮你完成一篇高质量公众号文章

[wechat-article-extractor](https://github.com/freestylefly/wechat-article-extractor-skill/wechat-article-extractor) 输入公众号链接，自动提取文章全部信息

[wechat-article-publisher](https://github.com/iamzifei/wechat-article-publisher-skill/wechat-article-publisher) 通过API将Markdown/HTML发布到公众号草稿箱

[auth-wechat-miniprogram](https://github.com/tencentcloudbase/skills/auth-wechat-miniprogram) 微信小程序云开发身份认证，无需复杂OAuth流程

[huashu-skills](https://github.com/alchaincyf/huashu-skills) 花叔的 Claude Code Skills 合集。21个经过实战验证的内容创作技能，覆盖从选题到发布的完整工作流
