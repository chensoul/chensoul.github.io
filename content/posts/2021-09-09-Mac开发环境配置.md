本文记录 **重装 macOS 前后** 的 checklist。日常 shell 与软件清单以私有仓库 **`dotfiles`** 及其 README 为准（文中命令与仓库若有出入，以仓库为准）；路径、邮箱、远程地址请按你的环境改写。

## 重装 macOS 之前

抹盘或「抹掉所有内容与设置」前，先确认**数据与授权**已在其他介质或云端有副本，避免重装后缺文件、缺密钥。

### 1. 备份文件与密钥（iCloud 云盘）

统一备份到 **iCloud 云盘**（访达侧栏「iCloud 云盘」与下列路径一致）。重装后同一 Apple ID 登录即可同步。

```bash
ICLOUD="$HOME/Library/Mobile Documents/com~apple~CloudDocs"
BACKUP="${ICLOUD}/backup/mac-reinstall"
mkdir -p "$BACKUP"
```

**合规**：公司代码、密钥是否允许进入 iCloud 须遵守你方政策；极敏感内容请用加密移动硬盘或密码管理器。

- **代码目录**（示例为 `~/github`、`~/work`）

  打包前可用 `cleanup` 清理常见构建目录与 `*.log`、`.DS_Store`（定义在 `~/.aliases`；`cleanup -n` 可先预览不删除）。若无该函数，可手动删大目录后再打包。

  ```bash
  cd ~
  cleanup
  zip -r "${BACKUP}/github.zip" ~/github
  zip -r "${BACKUP}/work.zip" ~/work
  ```

- **本地 Maven 仓库**（体积大，也可重装后重新下载依赖）

  ```bash
  zip -r "${BACKUP}/m2.zip" ~/.m2
  ```

- **GPG**（Git 签名、Maven Central 发布等）

  自动取密钥环中**第一把**私钥并导出，上传公钥到密钥服务器：

  ```bash
  cd "$BACKUP"
  KEY_ID=$(gpg --list-secret-keys --with-colons 2>/dev/null | awk -F: '/^sec:/{print $5; exit}') && \
  test -n "$KEY_ID" && \
  gpg --armor --export-secret-keys "$KEY_ID" > gpg-private-key.asc && \
  gpg --armor --export "$KEY_ID" > gpg-public-key.asc && \
  gpg --keyserver keys.openpgp.org --send-keys "$KEY_ID"
  ```

  文件保存在 **`${BACKUP}/`**，文件名含 `KEY_ID`。导出私钥时若提示口令，属正常。

- **Insomnia** 单元测试文件使用云服务备份


### 2. 应用配置导出

对无法仅靠重装恢复的软件提前导出，例如 **cc-switch**、**TablePlus**、**SwitchHosts**、**Clash Verge**（订阅与规则）等，将导出文件放入 **`${BACKUP}/`** 或子目录。**勿**将含密钥的导出提交到公开仓库。

### 3. 软件清单（Homebrew）

在「软件已装齐」的本机导出清单，便于对照：

```bash
git clone git@github.com:chensoul/dotfiles.git
cd dotfiles
brew bundle dump --force --file=Brewfile
git add .
git commit -m 'backup Brewfile'
git push origin main
```

### 4. 其他

- 密码管理器（如 1Password）确认 **主密码 / Secret Key** 在新环境可用。  
- **转手或出售**：退出 iCloud、App Store、iMessage 后再抹盘；若曾在 iCloud 存密钥或代码包，在其他设备上删除或移走敏感内容。

## 重装 macOS 之后

以下顺序兼顾 **先能连上 Git**，再 **装工具链**。

### 1. 系统与账号

登录 Apple ID，打开 **iCloud 云盘** 同步；设置时区、输入法；若公司要求再开 FileVault 等。

### 2. Command Line Tools

```bash
xcode-select --install
```

未完成前不要执行 `install.sh`（脚本内也会检查）。

### 3. 从 iCloud 恢复备份

同步完成后，备份目录一般为：

`~/Library/Mobile Documents/com~apple~CloudDocs/backup/mac-reinstall/`

```bash
ICLOUD="$HOME/Library/Mobile Documents/com~apple~CloudDocs"
BACKUP="${ICLOUD}/backup/mac-reinstall"

gpg --import "${BACKUP}"/gpg-private-key.asc 2>/dev/null || true
gpg --import "${BACKUP}"/gpg-public-key.asc 2>/dev/null || true

# 按需解压（路径按你备份时的 zip 名称）
unzip "${BACKUP}/m2.zip" -d ~
unzip "${BACKUP}/github.zip" -d ~
unzip "${BACKUP}/work.zip" -d ~
```

### 4. 克隆 dotfiles 并安装环境

```bash
git clone https://github.com/chensoul/dotfiles.git
cd dotfiles
chmod +x install.sh
bash install.sh
```

### 5. 应用与插件

用重装前的导出恢复 **cc-switch**、**TablePlus**、**SwitchHosts**、**Clash Verge** 等；无备份则只能手工重配。

**IntelliJ IDEA** 插件示例：`.ignore`、Atom Material Icons、Laconic POM for Maven、Maven Helper、Rainbow Brackets。  

**Chrome** 插件示例：JsonVue、1Password。

### 验收

```bash
java -version
node -v   # 若使用 fnm，按 dotfiles 中说明加载
git st
gpg -K    # 确认密钥已导入（若使用签名）
```

再 `git clone` 一个**私有仓库**，确认 SSH 与（若使用）代理正常。
