最近看到一篇文章《[Claude Code是最好的自动化写作Agent，附完整教程！](https://mp.weixin.qq.com/s/XvFZtoybx_u_W8Os4eQdlQ)》，对于其中搭建自动化写作 Agent 的想法感兴趣。

从作者公众号文章中找到了作者的 github 仓库[地址](https://github.com/alchaincyf)，接着找到了作者创作的 [Skills 合集](https://github.com/alchaincyf/huashu-skills) ，包括AI审校、选题生成、视频大纲、素材搜索等 11 个实用技能。另外，有一个仓库是[公众号排版器](https://github.com/alchaincyf/huasheng_editor)。

找到这些信息之后，就想参考里面的思路使用 Claude Code 来搭建一套自动化写作流程。

其实，也不局限于 Claude Code，也可以使用 OpenClaw 来搭建，操作过程是一样的。我先是在 Claude Code 里面跑通了整个流程，现在又在 OpenClaw 里面实践了一遍。

由于公众号文章有反爬机制，只能直接将文章内容复制发送给 openclaw，让 openclaw 进行总结。

openclaw 总结出这篇文章的核心价值：

1. 两层判断机制

```bash
第一层：工作区判断
├── 公众号写作？
├── 视频创作？
├── Prompt 梳理？
└── 其他？

第二层：任务类型判断
├── 新写作任务（有完整 brief）
├── 新写作任务（无 brief 只有需求）
├── 修改已有文章
├── 文章审校/降 AI 味
└── 快速咨询
```

2. 公众号写作 9 步流程

   - 理解需求 & 保存 Brief

   - 信息搜索与知识管理

   - 选题讨论（必做）

   - 创建协作文档（如需测试/配图）

   - 学习我的风格
   - 等待测试数据
   - 创作初稿
   - 三遍审校（降 AI 味）⭐️

   - 文章配图

3. 降 AI 味的核心方法

- 个人素材库：用真实经历、观点、案例替代 AI 腔
- 三遍审校：内容 → 风格 → 细节
- 风格学习：读取作者已有文章，提取特征

4. 最重要的设计理念：流程是指南，不是教条；核心原则不可妥协。



将  [Skills 合集](https://github.com/alchaincyf/huashu-skills) 发给 openclaw，openclaw 整理出了这个 Skills 合集的核心价值

覆盖的完整链路

```bash
输入 → 选题 → 调研 → 写作 → 审校 → 配图 → 发布 → 多平台分发
```

21 个 Skills 分类

📝 写作类

1. huashu-ai-cleanup - AI 审校（三遍审校，降 AI 味）
2. huashu-personal-material - 个人素材库检索
3. huashu-editor - 标准化编辑
4. huashu-topic-generator - 选题生成
5. huashu-research - 渐进式调研
6. huashu-knowledge-search - 新知识搜索

🎬 视频类

7. huashu-video-script - 视频脚本生成
8. huashu-video-analyzer - 竞品视频分析
9. huashu-video-outline - 视频大纲
10. huashu-video-oralize - 视频脚本口语化
11. huashu-mrbeast-check - MrBeast 风格检查

🎨 设计/配图类

12. huashu-design-brief - 设计需求澄清
13. huashu-article-illustrator - 文章配图
14. huashu-xiaohongshu-illustrator - 小红书配图
15. huashu-auto-illustrator - 自动配图 + 上传图床
16. huashu-slides - PPT 生成

📊 报告/数据类

17. huashu-data-report - 数据分析报告
18. huashu-whitepaper - PDF 白皮书生成

📱 多平台分发

19. huashu-x-platform - 公众号转 X 平台（微博/小红书）

🎤 演讲/协作

20. huashu-speak - 演讲准备
21. huashu-multi-agent - 多 Agent 并行协作

并且给出了安装建议：不需要一次性全装，先装最核心的 5 个：

第一阶段：核心写作能力（优先）

- huashu-ai-cleanup - AI 审校 ⭐️⭐️⭐️

- huashu-topic-generator - 选题生成 ⭐️⭐️⭐️

- huashu-personal-material - 个人素材库 ⭐️⭐️⭐️

- huashu-research - 渐进式调研 ⭐️⭐️

- huashu-article-illustrator - 文章配图 ⭐️⭐️

第二阶段：扩展能力（可选）

- huashu-editor - 标准化编辑
- huashu-knowledge-search - 新知识搜索
- huashu-x-platform - 多平台分发

第三阶段：视频/设计（以后再说）

- 视频类 Skills
- 设计类 Skills
- PPT/白皮书

这些技能也不一定只能在 claude code 中使用，我问 openclaw 可以在 openclaw 中使用吗。

openclaw 回答 Claude Code Skills 本质上是结构化的提示词 + 工具调用规则，这些核心逻辑是可以迁移到 OpenClaw 的。于是，开始让 openclaw 进行迁移。
