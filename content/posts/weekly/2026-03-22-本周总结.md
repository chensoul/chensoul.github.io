#### 6、Linkding 技能接入

1. **安装** [clinkding CLI](https://github.com/daveonkels/clinkding)

通过 Homebrew 安装（macOS/Linux）：

```bash
brew install daveonkels/tap/clinkding
```

或通过 Go 安装：

```bash
go install github.com/daveonkels/clinkding@latest
```

2. **配置 Linkding 连接**

获取 Linkding 实例信息：

- URL：https://linkding.chensoul.cc
- API Token：从 Linkding Web 界面生成

配置方式一：交互式初始化

```bash
clinkding config init
```

配置方式二：手动配置

```bash
clinkding config set url https://linkding.chensoul.cc
clinkding config set token YOUR_API_TOKEN
```

配置方式三：环境变量

```bash
export LINKDING_URL="https://linkding.chensoul.cc"
export LINKDING_TOKEN="your-api-token"
```

配置文件位置：~/.config/clinkding/config.yaml

3. **测试连接**

```bash
clinkding config test
```

4. **技能配置**

技能位置：`~/.openclaw/workspace/skills/clinkding/SKILL.md`

5. **默认设置**

**根据我的个人偏好，我告诉小龙虾每次调用 clinkding 技能保存书签时**：

- 默认使用 --shared=true 参数，保存为公开书签
- 默认使用 --unread=true 参数，保存为未读书签

## 本周分享

虽然大部分有意思的输入都会自动同步到「[ChenSoul Share](https://t.me/s/ichensoul)」Telegram 频道，但我还是想在这里额外挑一部分出来。单独列出来之后，这篇周报就更像一份 newsletter 了。


上面这些内容，是我通过脚本从 [我的书签](https://linkding.chensoul.cc) 里统计出来的本周分享。

```bash
# 使用 jq 输出上周分享的 Markdown 链接
start=$(date -v-1w -v-mon +%Y-%m-%dT00:00:00Z)
end=$(date -v-mon +%Y-%m-%dT00:00:00Z)

curl -sS -H "Authorization: Token ${LINKDING_TOKEN}" \
  "https://linkding.chensoul.cc/api/bookmarks/?limit=100" \
  | jq -r --arg start "$start" --arg end "$end" '
      .results[]?
      | select(.date_added >= $start and .date_added < $end)
      | "- [" + (.title // "无标题") + "](" + .url + ")"
    '
```
