互联网已成为我们日常生活的重要组成部分，塑造着我们如何沟通、获取信息和开展业务。在其核心，互联网是使用标准化通信协议促进数据交换的互连计算机网络的全球系统。这使得文本、图像、视频等能够在各种设备间传输。

在本期中，我们深入探讨互联网的本质，探索其关键组件：网络边缘、接入网络、网络核心、网络协议和互联网协议栈。我们讨论分组交换、转发和路由如何工作。我们揭开接入网络的复杂性，并检查协议在管理互联网活动中发挥的关键作用。结束时，你将全面理解互联网架构及其在现代通信中的关键作用。

## 互联网的关键组件

互联网是复杂且互连的网络，由几个关键组件组成。我们将探索互联网最重要的组件。

### 网络边缘（Network Edge）

连接到互联网和其他设备通常称为端系统。它们称为端系统是因为它们位于互联网的"边缘"。

互联网的端系统包括台式电脑、服务器、移动设备，以及越来越多的非传统"物"，如智能家电和 IoT 设备正作为端系统连接到互联网。

端系统也称为**主机**，因为它们托管应用程序，如 Web 浏览器、Web 服务器、电子邮件客户端或电子邮件服务器。

主机有时进一步分为两类：**客户端**和**服务器**。非正式地，客户端倾向于台式和移动 PC、智能手机和类似个人计算设备，而服务器倾向于更强大的机器，存储和分发 Web 页面、流媒体视频、中继电子邮件和类似服务。今天，大多数提供搜索结果、电子邮件、网页和视频的服务器位于大型数据中心。

### 接入网络（Access Network）

在考虑了"网络边缘"的应用和端系统后，让我们接下来考虑接入网络，它是物理连接端系统到第一个路由器（也称为"边缘路由器"）的网络，在从端系统到任何其他遥远端系统的路径上。

接入网络作为端系统和更广泛网络基础设施之间的关键链接。接入网络可以广泛分为三种类型。

#### 1. 家庭接入网络（Home Access Networks）

指使住宅环境内连接和通信成为可能的技术集。此网络允许家庭内设备连接到互联网、共享数据并相互通信。

#### 2. 机构接入网络（Institutional Access Networks）

指组织、机构和公司用于连接到互联网并促进其场所内通信的网络基础设施和技术。这些网络设计用于处理大规模运营的特定需求和需求。

#### 3. 移动接入网络（Mobile Access Networks）

指使移动设备（如智能手机和平板电脑）连接到互联网并相互通信的各种技术。这些技术促进移动用户的无线通信和数据传输。

## 译者总结

这篇文章涵盖了互联网的核心概念：

**互联网定义**：
- 全球互连计算机网络系统
- 使用标准化协议促进数据交换
- 传输文本、图像、视频等

**关键组件**：
| 组件 | 说明 |
|------|------|
| 网络边缘 | 端系统（主机）位于网络边缘 |
| 接入网络 | 连接端系统到边缘路由器 |
| 网络核心 | （本文未详细覆盖） |
| 网络协议 | 管理互联网活动 |
| 协议栈 | 互联网协议层次 |

**端系统分类**：
| 类型 | 示例 |
|------|------|
| 客户端 | 台式 PC、移动 PC、智能手机 |
| 服务器 | 更强大机器，存储和分发内容 |

**接入网络 3 种类型**：
1. **家庭接入网络**：住宅环境连接
2. **机构接入网络**：组织/企业网络
3. **移动接入网络**：移动设备无线通信

**关键洞察**：
- 互联网是复杂互连网络
- 端系统位于网络边缘
- 接入网络是关键链接
- 三种接入类型满足不同需求

理解这些概念对理解互联网架构至关重要。
