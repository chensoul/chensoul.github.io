## 不同类型测试的位置

编写代码现在很容易，但测试代码很难。

让我们看看不同类型的测试适合哪里：

### 单元测试 + 组件测试
这些测试隔离的单个函数或 UI 组件。它们快速、运行成本低且易于维护。像 Jest、Vitest、JUnit、pytest、React Testing Library、Cypress、Vue Test Utils 和 Playwright 这样的工具通常用于这里，你的大部分测试覆盖应该来自这一层。

### 集成测试
这些验证服务、API 和数据库之间的通信。Testcontainers、Postman、Bruno、Supertest。单元测试不会捕获损坏的 API 合同，但集成测试会。

### 端到端测试
像 Cypress、Playwright、Appium 和 QA Wolf 这样的工具验证整个系统的完整用户旅程。它们运行和维护成本高，这就是为什么更少的测试生活在这一层。

AI 工具正在成为测试工作流的一部分。像 GitHub Copilot、ChatGPT、Claude、Cursor 和 Qodo 这样的工具可以帮助起草测试、更新套件并发现覆盖范围中的差距。它们处理重复任务，让工程师有更多时间关注生产中可能出现的边缘情况。

## 单点登录（SSO）如何工作

单点登录（SSO）使访问感觉轻松。一次登录，你就进入 Slack 和几个其他内部工具，无需再次登录。

但这背后有很多事情发生。

### 步骤 1：第一次登录

- 用户打开应用，例如 Salesforce
- Salesforce 不是直接要求凭据，而是将浏览器重定向到身份提供商（IdP），如 Okta 或 Auth0。这个重定向通常通过 HTTP 302 响应发生
- 然后浏览器使用 SAML 或 OpenID Connect（OIDC）等协议向 IdP 发送认证请求
- IdP 显示登录页面。用户输入凭据，有时连同 MFA
- 一旦验证，IdP 创建登录会话并通过浏览器发回认证响应（SAML 断言或 ID 令牌）
- 浏览器将该响应转发回 Salesforce
- Salesforce 验证令牌并创建自己的本地会话（通常存储为 cookie），并授予访问权限

### 步骤 2：SSO 魔法

- 现在用户打开另一个应用，比如 Slack
- Slack 也将浏览器重定向到相同的身份提供商。但 IdP 检查并看到用户已经有活动会话。所以它完全跳过登录步骤并发行新的认证令牌
- 浏览器将该令牌转发到 Slack
- Slack 验证它，创建自己的会话 cookie，并授予访问权限

SSO 背后的关键想法很简单。应用不自己认证用户。它们依赖中央身份提供商验证用户并发行其他系统信任的令牌。

## LLM 如何使用 AI 代理进行深入研究

当你要求 LLM（如 Claude、ChatGPT 或 Gemini）对复杂主题进行深入研究时，不只是一个模型在做所有工作。它是一个协调的专门 AI 代理系统。

### 步骤 1：理解问题和制定计划

一切都从查询开始，比如"分析 2026 年 AI 代理的竞争格局"。系统不是盲目深入。首先，它可能会问澄清问题以确切理解需要什么。然后，它生成计划，将大问题分解为更小、可管理的任务。

### 步骤 2：子代理开始工作

每个小任务分配给子代理，它基本上是有特定工作的迷你 AI 工作者。例如，一个子代理可能 tasked 查找最新的 Nvidia 收益。它找出使用哪些工具，如搜索网络、浏览特定页面，甚至运行代码分析数据。所有这些都通过连接 AI 到外部世界的安全 API 和服务层发生。

### 步骤 3：整合

一旦所有子代理完成它们的任务，合成代理接管。它聚合一切，识别关键主题，计划大纲，并移除任何冗余或重复信息。同时，引用代理确保每个声明链接回其来源并正确格式化。最终结果是抛光、良好引用的最终输出，准备使用。

## 黑客如何窃取密码

大多数密码攻击不涉及复杂黑客。它们依赖自动化、重用的凭据和可预测的人类行为。

以下是六种常见技术：

### 1. 暴力攻击
自动化工具高速循环密码组合直到一个有效。不涉及逻辑，只是数量。

### 2. 字典攻击
不是随机猜测，攻击者使用从常见密码、泄露数据和可预测模式构建的精选词表。

### 3. 凭据填充
当一个网站被泄露时，攻击者将这些被盗的用户名 - 密码对重用到许多其他服务。它有效因为很大一部分用户在多个账户上重用密码。

### 4. 密码喷洒
一个常见密码在同一组织中的许多账户上尝试。跨账户分散尝试避免触发锁定阈值。

### 5. 网络钓鱼
受害者落在假登录页面并输入凭据。攻击者实时捕获它们。不需要恶意软件。

### 6. 键盘记录恶意软件
恶意软件记录击键并将它们发送给攻击者。密码、用户名以及在设备上输入的任何其他内容都被捕获。
